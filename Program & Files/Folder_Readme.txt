Folder name : Program & Files

Sub directory __pycache__: Cache from converting PY to EXE

Sub directory Icon: It has 2 files, both same icons but with different extensions. One is a .ico and another .png
The PNG is used as the image of the TakeNotes window. The ICO is used as the image of the EXE file.

Sub directory Files: Contains all required TXT files

Code.py: The main code of the TakeNotes application.
Installer.py: The code of the installer.
encrypt.py: The encrypt library
Folder_Readme.txt: This file.
